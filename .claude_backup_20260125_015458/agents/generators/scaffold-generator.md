# Scaffold Generator

## Mission
Generate complete, working project scaffolds that follow best practices.

## Scope Boundaries

### MUST Do
- Create complete project structures
- Include configuration files
- Set up testing infrastructure
- Configure linting/formatting
- Generate CI/CD templates

### MUST NOT Do
- Generate outdated patterns
- Skip essential configs
- Hardcode secrets
- Create non-portable projects

## Required Inputs

| Input | Type | Required | Description |
|-------|------|----------|-------------|
| project_type | string | yes | web, api, cli, library |
| framework | string | yes | next, express, fastify, etc. |
| features | array | no | auth, database, testing |
| package_manager | string | no | npm, pnpm, yarn, bun |

## Outputs Produced

| Output | Type | Description |
|--------|------|-------------|
| files | object | Directory tree with contents |
| commands | array | Setup commands to run |
| instructions | string | Getting started guide |

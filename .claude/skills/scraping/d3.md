## 3. scaleSqrt Replacement Checklist

### File to Modify

#### 3.1 `/src/lib/workers/force-simulation.worker.ts`

**Line 9: Remove import**
```typescript
// DELETE:

```

**Add after imports (around line 50):**
```typescript
/**
 * Native square root scale - replaces d3-scale.scaleSqrt()
 */
const createSqrtScale = (value: number, maxValue: number, rangeMin: number, rangeMax: number): number => {
  if (maxValue === 0) return rangeMin;
  const t = Math.sqrt(value / maxValue);
  return rangeMin + t * (rangeMax - rangeMin);
};
```

**Line 294: Replace scale creation**
```typescript
// BEFORE

// AFTER (remove this line, radiusScale will be called differently)
```

**Line 313: Update scale usage**
```typescript

// AFTER

```

WHAT'S THE PROBLEM?
==================

Blank white screen for 250-350ms before loading screen appears.

Root cause: onMount() blocks initialization, delaying loading state update.

Expected fix: 250ms improvement (60-70% faster FCP).

IMPLEMENTATION SUMMARY
=======================

In data.ts:
  Before: async initialize() { await ... }
  After:  initialize() { import(...).then(...) }
  Result: Returns immediately, updates in background

In +layout.svelte onMount():
  Before: Promise.allSettled([await init1, await init2, ...])
  After:  status.set('loading'); Promise.allSettled([init1, init2, ...])
  Result: Loading state set before any awaits, immediate render

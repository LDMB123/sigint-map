================================================================================
                            APPLE SILICON WINS
================================================================================

Optimization: Metal Backend + UMA Zero-Copy Buffers
  Technique: mappedAtCreation: true
  Benefit: Eliminate CPU↔GPU memory copy (30-50% latency reduction)
  Code:
    const buffer = device.createBuffer({
      size: data.byteLength,
      usage: GPUBufferUsage.STORAGE,
      mappedAtCreation: true  // Zero-copy on Apple Silicon
    });

Optimization: Optimal Workgroup Size for Apple GPU
  Setting: @workgroup_size(256) = 32 × 8 waves
  Benefit: Perfect occupancy on Apple GPU (15-25% speedup)
  Code:
    @compute @workgroup_size(256)
    fn main(@builtin(global_invocation_id) id: vec3<u32>) {
      // 256 threads optimal for Apple hardware
    }

Optimization: Native Float16 Support
  Feature: shader-f16 available on M-series
  Benefit: 50% memory bandwidth reduction (20-40% faster)
  Code:
    if (adapter.features.has('shader-f16')) {
      requiredFeatures.push('shader-f16');
    }

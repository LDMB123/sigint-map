CORE OPTIMIZATION CONCEPT
================================================================================

scheduler.yield() is a Chromium 129+ API that yields to the browser event loop:

BEFORE: Large batches
  Process 250 items (250ms) → Yield → Process 250 items (250ms)
  User input blocked for 250ms → INP = 250ms ✗

AFTER: Small batches with yields
  Process 50 items (50ms) → Yield → Process 50 items (50ms) → Yield → ...
  User input processed during yields → INP = 50ms ✓

Benefit: 50% faster than setTimeout(0) on Chromium
Apple Silicon: E-cores handle background during P-core yield

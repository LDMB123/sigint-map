## DevTools Memory Profiling Workflow

### Step-by-Step Process

```
Chrome DevTools Memory Panel Workflow
в•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җв•җ

1. Open DevTools
   в”ңв”Җ Cmd+Option+I (Mac) or F12 (Windows)
   в””в”Җ Go to "Memory" tab

2. Take Initial Snapshot
   в”ңв”Җ Click "Heap snapshot"
   в”ңв”Җ Click "Take snapshot"
   в””в”Җ Note: Initial heap size = ~80-100 MB


3. Perform Suspected Leaking Action
   в”ңв”Җ For rate limit: make 1000 requests
   в”ңв”Җ For WASM: run heavy operations 100 times
   в”ңв”Җ For observer: scroll page up/down 10 times
   в””в”Җ For tracker: complete 500+ operations


4. Force Garbage Collection
   в”ңв”Җ Click trash icon (top left)
   в”ңв”Җ Wait 1-2 seconds
   в””в”Җ Any persistent objects = likely leak


5. Take Second Snapshot
   в”ңв”Җ Click "Take snapshot" again
   в”ңв”Җ Note: Final heap size
   в””в”Җ Calculate difference


6. Compare Snapshots
   в”ңв”Җ Click "Comparison" dropdown
   в”ңв”Җ Select first snapshot
   в”ңв”Җ Look for "+500MB" or similar
   в”ңв”Җ Check object counts:
   в”Ӯ  в”ңв”Җ EventListener: should be same
   в”Ӯ  в”ңв”Җ Map: should be same
   в”Ӯ  в”ңв”Җ Array: check for growth
   в”Ӯ  в””в”Җ HTMLElement: check for detached nodes
   в””в”Җ Filter by "Detached" to find orphaned DOM


7. Analyze Retainers
   в”ңв”Җ Click on suspicious object
   в”ңв”Җ See "Retainers" panel
   в”ңв”Җ Trace back to root object
   в”ңв”Җ Example: Detached DIV вҶ’ kept alive by:
   в”Ӯ  в””в”Җ IntersectionObserver closure вҶ’ Window global
   в””в”Җ Fix: add null checks and cleanup


8. Before/After Verification
   Without Fix:
   в”ңв”Җ Initial: 80 MB
   в”ңв”Җ After action: 200 MB
   в”ңв”Җ After GC: 180 MB вң— (leak!)
   в””в”Җ Difference: +100 MB leak confirmed

   With Fix:
   в”ңв”Җ Initial: 80 MB
   в”ңв”Җ After action: 95 MB
   в”ңв”Җ After GC: 85 MB вң“ (no leak!)
   в””в”Җ Difference: +5 MB normal objects

```

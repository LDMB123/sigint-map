## Key Innovations

### 1. Synchronous Guard Flag Pattern
```typescript
// Problem: Async initialization allows race conditions
// Solution: Set guard synchronously BEFORE async work
let initialized = false;
initialize() {
  if (initialized) return;        // ← Set BEFORE async
  initialized = true;
  // ... then do async work
}
```

### 2. Exponential Backoff with Max Cap
```typescript
// Problem: Linear backoff wastes resources
// Solution: Exponential (1s → 1.5s → 2.25s...) capped at 60s
const backoffMs = Math.min(
  item.backoffMs * Math.pow(1.5, item.retryCount),
  60000
);
```

### 3. MessageChannel Port Cleanup in All Paths
```typescript
// Problem: Ports left open → memory leak
// Solution: Close ports in success, timeout, AND error paths
const port = new MessageChannel();
const timeout = setTimeout(() => {
  port.port1.close();  // ← Must close in all branches
  resolve(null);
});
port.port1.onmessage = () => {
  port.port1.close();  // ← Close on success
};
port.port1.onerror = () => {
  port.port1.close();  // ← Close on error
};
```

### 4. Dexie.js + Background Sync Integration
```typescript
// Problem: Offline queue doesn't persist across reloads
// Solution: IndexedDB + Background Sync + Fallback
export const db = new Dexie('sync-db');
db.queue.put(entry);
// + Service Worker processes on:
// - Online event
// - Background Sync (if supported)
// - Periodic Sync (if enabled)
```

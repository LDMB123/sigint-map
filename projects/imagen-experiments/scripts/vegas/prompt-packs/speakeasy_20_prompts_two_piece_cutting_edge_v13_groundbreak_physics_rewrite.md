# Speakeasy Prompt Pack v13 - Groundbreak Physics Rewrite

## Objective
Create photoreal, identity-faithful, lens-locked editorial imagery using a physics-first prompt system that outperforms texture-only prompting.

## Innovation Architecture
This pack is a full rewrite of all prompts using a constraint graph:
- Identity Node: fixed facial geometry and expression asymmetry.
- Gaze Node: strict lens-axis eye alignment and catchlight coherence.
- Scene Node: named anchor objects with domain lock (commercial nightlife only).
- Wardrobe Node: mandatory two-piece replacement topology with support stability.
- Physics Node: force-chain closure from load -> strain -> fold -> highlight -> shadow.
- Safety Node: non-explicit cocktail-fashion framing with unambiguous coverage-critical edges.

## QA Contract (for every prompt)
All outputs must satisfy:
1. Identity fidelity to reference face geometry.
2. Direct lens-centered gaze.
3. Full wardrobe replacement to prompt attire.
4. At least 3 named scene anchors visibly present.
5. Physics causality at 100% zoom (no painted folds/specular artifacts).
6. Safety-compliant, non-explicit editorial framing.

## Operational Notes
- Keep adult subject only (25+).
- No bedroom, home, or residential substitutions.
- No one-piece substitution when two-piece is specified.
- If any hard lock is violated, regenerate.

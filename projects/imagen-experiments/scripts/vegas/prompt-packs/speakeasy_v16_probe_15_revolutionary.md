## Prompt 15 - Neon Booth Refraction

### Primary Prompt

MISSION:
Create a photoreal nightlife editorial image in window booth overlooking neon street signage through clean glass. The output must feel like premium campaign photography, never synthetic artwork.

IDENTITY + GAZE LOCK:
- Use reference image only for the same adult woman (25+).
- Preserve face geometry: orbital spacing, brow architecture, nose bridge, lip contour, jawline, and natural asymmetry.
- Keep strict lens-axis eye contact with physically plausible vergence and coherent catchlights.
- Hard fail if identity drifts or gaze leaves lens axis.

SCENE DOMAIN LOCK:
- Required visible anchors: clean glass panel, neon OPEN reflection, leather booth backrest, table candle practical.
- Keep upscale commercial speakeasy environment only.
- Never introduce home/bedroom/residential elements.
- Keep scene dry and interior-focused with no liquid/vapor atmospheric effects.

WARDROBE TOPOLOGY LOCK:
- Full replacement attire: black crystal two-piece set with sculpted top and moderated slit evening skirt, black sheer hosiery, crystal pumps.
- Enforce true two-piece topology: separate top + separate skirt with visible waist boundary.
- Prohibit one-piece merges, bodysuit fallback, denim/casual carryover, or unresolved garment boundaries.

POSE + STORY:
- Pose directive: table-edge stand with one hand on tabletop and one at hip.
- Narrative directive: closing-time command shot.
- Expression must read confident, composed, and editorial.

PHYSICS MATRIX:
- Signature phenomenon: layered glass refraction parallax and crystal highlight conservation.
- Force/load path: table support -> shoulder set -> waist lock -> drape equilibrium.
- Mechanical tests that must pass:
- Glass refraction must shift background neon with correct directionality and perspective.
- Crystal embellishments must sparkle selectively, not uniformly across panel.
- Table-hand pressure should produce slight skin flattening and tendon cues.
- Booth leather specular must remain broad and roughness-consistent.

CAMERA + LIGHT SPEC:
- Camera rig: vertical 4:5, 52mm full-frame look, f/2.6, full-body interior/exterior layered depth.
- Lighting rig: candle key with cool street-neon ambient through glass and soft interior fill.
- Enforce coherent key/fill/rim causality and shadow direction discipline.

REJECT MATRIX:
- Reject if exterior reflections ignore interior camera angle.
- Reject if candle key creates contradictory double shadows.
- Reject if slit hem intersects leg geometry unnaturally.
- Reject if liquid streaks, fogging, or wet-surface artifacts appear.
- Reject any explicit framing, nudity, fetish coding, anatomy break, texture repetition, clipping, or plastic skin.

# Speakeasy Prompt Pack v15 - Physics Matrix Extreme

## Core Innovation
Complete rewrite driven by a per-prompt physics validation matrix, not only style descriptors.

## Operating Model
Every prompt now includes:
- Identity Lock: strict face geometry continuity from reference.
- Lens Lock: direct lens-axis gaze with catchlight coherence.
- Domain Lock: required nightlife anchors, no residential substitution.
- Garment Topology Lock: mandatory two-piece replacement with visible waist break.
- Physics Matrix: signature phenomenon, explicit load path, and 4 hard mechanical tests.
- Reject Matrix: deterministic fail conditions.

## Global Hard Rules
1. Adult subject only (25+).
2. Non-explicit premium editorial framing only.
3. No one-piece substitution when two-piece is required.
4. Full-body readability in vertical 4:5.
5. Regenerate on any identity, gaze, topology, or physics failure.

## Boundary Findings (Compressed)

**Source:** BOUNDARY-FINDINGS-REPORT.md (20 KB)

### What Works

**100% success rate:**
- Camera sensor physics (ISO, noise patterns, JPEG artifacts)
- Real location names (Tootsie's Orchid Lounge, specific venues)
- Brand specificity (Tony Lama boots, PBR tallboy)
- Wear patterns (scuffs, wrinkles, makeup breakdown)
- Technical photography language

**75-90% success rate:**
- Light transport equations
- Fabric stress analysis
- Skin subsurface scattering
- Material BRDF descriptions

### What Doesn't Work

**Immediate rejection:**
- Generic descriptions without physics
- Perfect/idealized elements
- Vague lighting ("soft lighting")
- Missing technical anchors
- Short prompts (<400 words)

### Critical Thresholds

- **Minimum viable:** 600 words
- **Optimal range:** 700-900 words
- **Diminishing returns:** >1000 words
- **Physics ratio:** 70%+ required

## Implementation Patterns

### Successful Prompt Structure

```
1. Composition (50-80 words)
   - Rule of thirds
   - Frame fill percentage
   - Eye contact direction

2. Camera Physics (120-150 words)
   - Sensor type, ISO, aperture, shutter
   - Noise characteristics
   - JPEG artifacts
   - Lens aberrations

3. Scene Context (60-80 words)
   - Real location name
   - Specific time
   - Environmental details
   - Authentic moment description

4. Attire & Materials (100-120 words)
   - Fabric physics
   - Wear patterns
   - Material behavior under stress

5. Light Transport (150-180 words) ← CRITICAL
   - Multiple source specification
   - Color temperatures
   - Surface interactions
   - Shadow/highlight behavior

6. Skin Detail (100-120 words)
   - Subsurface scattering
   - Pore variation by region
   - Natural imperfections
   - Late-night characteristics

7. Imperfection Anchors (120-150 words)
   - Wear, wrinkles, stains
   - Makeup breakdown
   - Hair disorder
   - Real-world flaws
```

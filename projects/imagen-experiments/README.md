# Imagen Experiments

AI image generation experiments using Google Imagen, Veo, and other generative AI tools.

## Project Structure

```
imagen-experiments/
├── scripts/          # Generation scripts (shell & JavaScript)
├── prompts/          # Concept files and prompts
└── assets/           # Reference images
```

## Scripts

- **Imagen scripts**: `imagen-*.js` - Direct Imagen API integration
- **Veo scripts**: `veo-*.js` - Video generation with Veo
- **Batch scripts**: `generate-*.sh`, `*-batch.sh` - Batch processing
- **Concept generation**: Scripts for generating themed content

## Usage

See individual script files for usage instructions.

## Generated Content

This is a personal AI art generation project for experimenting with:
- Fashion photography concepts (swimwear, editorial)
- Themed concept generation (dive bar series)

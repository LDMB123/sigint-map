## Skill Details

### 1. js-to-css-audit.md
**File Size:** ~12KB | **Sections:** 12 | **Examples:** 8

Comprehensive guide for auditing codebases to identify JavaScript patterns that can be replaced with modern CSS features.

**Key Sections:**
- Pattern discovery and categorization
- Impact analysis and prioritization
- Conversion templates for 5+ common patterns
- Migration report generation
- Browser compatibility with fallbacks

**Best For:**
- Reducing JavaScript bundle size
- Improving animation performance
- Identifying quick wins in codebase

**Performance Example:**
- Remove IntersectionObserver + useState hooks
- Replace with `animation-timeline: view()`
- Saves 15-20 lines of JavaScript per component

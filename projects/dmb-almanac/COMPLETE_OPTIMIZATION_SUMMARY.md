## Understanding Agents vs Skills

### Agents (Executable)
- **What**: AI agents that execute tasks
- **Have**: Model tier assignments (Opus/Sonnet/Haiku)
- **Cost**: Token usage based on tier
- **Optimized**: ✅ Tier assignments + category consolidation

### Skills (Knowledge)
- **What**: Documentation, patterns, best practices
- **Have**: No model tier (they're reference material)
- **Cost**: No direct cost (used by agents when needed)
- **Optimized**: ❌ Already optimal (no changes needed)

**Example:**
```
Agent: rust-debugger (Sonnet tier)
  ↓ references
Skill: rust/borrow-checker-debug.md (knowledge)
```

The **agent** executes on Sonnet and may **reference** the skill for guidance.

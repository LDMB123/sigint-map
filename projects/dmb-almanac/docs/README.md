## Project Overview

The DMB Almanac is a Progressive Web App for Dave Matthews Band fans featuring:
- Complete show database (2,800+ shows)
- Setlist analysis and statistics
- Guest appearance tracking
- Offline-first architecture
- Chromium 143+ optimizations

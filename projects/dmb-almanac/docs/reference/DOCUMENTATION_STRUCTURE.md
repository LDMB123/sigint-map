## File Type Guide

### Markdown (.md) Files
- Human-written documentation
- Implementation guides
- Technical references
- Use for reading and learning

### Text (.txt) Files
- Machine-generated reports
- Audit summaries
- Quick references
- Use for quick scanning

## Naming Convention

All files follow this pattern:
```
[CATEGORY]_[DESCRIPTION]_[TYPE].[ext]

Examples:
BUNDLE_ANALYSIS_REPORT.md
MEMORY_LEAK_PREVENTION_GUIDE.md
WASM_CONVERSION_QUICK_REFERENCE.md
```

## Search Tips

### Find all documentation about a topic
```bash
cd docs/
grep -r "search term" **/*.md
```

### List all files in a category
```bash
ls -lh docs/bundle/
```

### Find recently modified files
```bash
find docs/ -type f -mtime -7 -name "*.md"
```

### Count total documentation
```bash
find docs/ -type f \( -name "*.md" -o -name "*.txt" \) | wc -l
```

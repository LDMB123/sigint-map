================================================================================
WHAT THE SCRAPER DOES
================================================================================

The releases scraper extracts DMB discography data:

Step 1: Discovers Releases
   - Loads discography list page
   - Finds all release links using CSS selectors
   - Expected: 40-50 releases found

Step 2: Parses Release Details
   - Title (from h1 tag)
   - Release type (studio, live, compilation, video, box_set, ep, single)
   - Release date (ISO format: YYYY-MM-DD)
   - Cover art URL
   - Complete track listing

Step 3: Extracts Track Information
   - Song title (linked to SongStats pages)
   - Track number and disc number
   - Duration (MM:SS format)
   - Source show date (for live albums)

Step 4: Respects Rate Limits
   - 2 concurrent requests maximum
   - 30 requests per minute maximum
   - 1-3 second delays between requests

Step 5: Manages Progress
   - Checkpoints every 10 releases
   - Caches HTML to avoid redundant fetches
   - Can resume if interrupted

Step 6: Saves Results
   - Output: output/releases.json
   - Format: Structured JSON with metadata
   - Ready for database import

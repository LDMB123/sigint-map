# Brand System V2 - DMB Almanac

## Brand Positioning
The living archive of DMB performances: trusted, fast, and warm.

## Brand Values
- Accuracy
- Warmth
- Community
- Energy
- Longevity

## Personality
Knowledgeable, not academic. Precise, not cold. Passionate, not hype-driven.

## Tone of Voice
- Use concrete dates and factual phrasing.
- Avoid excessive adjectives.
- Example: "Last played June 24, 2023."

## Visual System

### Color Palette
| Color | Hex | Usage |
| --- | --- | --- |
| Primary Ink | #1B1B17 | Primary text |
| Warm Amber | #D07B3B | CTAs, highlights |
| Deep Teal | #2F5E6E | Links, headers |
| Forest Green | #3C7A5B | Positive states |
| Rust | #C84D4D | Warnings and critical |
| Fog | #F3F1ED | Page background |
| Stone | #6B6B62 | Muted text |
| Night | #0F1E22 | Hero backgrounds |

### Typography
- Display: Fraunces (headlines, large numbers)
- Body: Atkinson Hyperlegible (body, UI)
- Mono: JetBrains Mono (code, ids)
- Fallback: ui-sans-serif, system-ui

### Type Scale
- Display XL: 3rem
- Display L: 2.25rem
- Body L: 1.125rem
- Body M: 1rem
- Caption: 0.75rem

### Layout Principles
- Strong typographic hierarchy.
- Summary strips aligned and readable.
- Limit shadows to functional emphasis.

### Grid & Spacing
- 8px base spacing system (4/8/12/16/24/32/48).
- Maximum content width: 1280px.
- Summary strips align to a consistent 12-column grid.

### Iconography
- Line icons, consistent stroke weight.
- Avoid decorative icons that reduce clarity.

### Motion
- Subtle transitions only.
- Prefer CSS scroll-driven effects for context.
- Motion should support comprehension, not decoration.

### Accessibility
- Minimum contrast ratio 4.5:1 for body text.
- Avoid low contrast decorative overlays.
- Ensure all interactive elements have focus state.

## Component Styling Rules
- Cards use subtle elevation only when interactive.
- Summary strips use all-caps labels with generous letter spacing.
- Buttons: warm amber primary, ghost secondary.
- Badges: solid fill for status, outline for metadata.

## Photography & Imagery
- Prefer warm, low-saturation imagery.
- Avoid high-contrast neon or cold blues.
- Use grain textures sparingly for depth.

## Logo Guidance
- Wordmark should be readable at 24px.
- Icon should evoke archive or light-beam motif.
- Clear space equals height of logomark.

## Do and Do Not

Do
- Use summary strips consistently.
- Keep copy short and factual.

Do Not
- Overuse accent colors.
- Use heavy gradients that reduce legibility.

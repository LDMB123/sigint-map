ISSUE #1: EMPTY VENUE NAMES (99 SHOWS) - SEVERITY: HIGH
================================================================================

Problem:
  - 99 shows have venueName field as empty string ""
  - Import script likely requires venue name for foreign key lookup
  - Results in import validation failure

Examples:
  ID: 453056860 | Date: 1991-01-01 | City: but he did not guest at this...
  ID: 453092229 | Date: 1991-01-01 | City: Charlottesville | State: VA
  ID: 453079529 | Date: 1991-01-01 | City: Charlottesville | State: VA
  ID: 453057022 | Date: 1991-01-01 | City: Charlottesville | State: VA
  ID: 453092143 | Date: 1991-01-01 | City: Charlottesville | State: VA

Data Structure Example:
  {
    "originalId": "453056860",
    "date": "1991-01-01",
    "venueName": "",                    // MISSING - CRITICAL
    "city": "but he did not guest...",  // CORRUPTED DATA
    "state": "in",                      // INCOMPLETE STATE CODE
    "country": "USA",
    "tourYear": 1991,
    "setlist": [],
    "guests": []
  }

Likely Cause:
  - Scraper failed to extract venue name from page
  - Venue reference not found in source data
  - Page parsing error for early DMB shows (1991)

Impact on Import:
  - Database import script checks for non-null venue_id
  - Foreign key constraint: venue_id INTEGER NOT NULL REFERENCES venues(id)
  - Without valid venue, record cannot be inserted

ISSUE #2: NO SETLIST DATA (646 SHOWS) - SEVERITY: MEDIUM
================================================================================

Problem:
  - 646 shows have empty setlist arrays []
  - Represents 17.1% of all shows
  - Missing song/setlist data means incomplete concert records
  - These records may also be marked for exclusion in import

Detail:
  - Shows with setlist: 3,126 (82.9%)
  - Shows without setlist: 646 (17.1%)
  - Most 1991 shows lack setlist data (source data not available)

Impact:
  - Users cannot see what songs were played
  - Song statistics incomplete
  - Rare/unique setlist information lost

Pattern by Year:
  1991: ~71 shows, mostly without setlist or venue details
  1992+: Better coverage, more shows have setlist data
  
Likely Cause:
  - Historical data (1991-1993) incomplete from source
  - Scraper wasn't retroactively collecting setlist data
  - Website may not have historical setlist data before certain year

ISSUE #3: MALFORMED CITY/STATE FIELDS (~50 SHOWS) - SEVERITY: HIGH
================================================================================

Problem:
  - Some shows have corrupted city field containing venue notes
  - State codes incomplete or invalid
  - Makes location lookup impossible

Example:
  Show ID: 453056860
  Actual city field value: "but he did not guest at this show.DMB was likely 
                           the first band to perform. Others in the lineup"
  
  This is NOT a city - it's text from a notes field or parsing error

Other Examples Found:
  - State: "in" (should be "IN" for Indiana or null)
  - Inconsistent state formatting
  - Notes text bleeding into city field

Data Pattern:
  Date: 1991-01-01 (very early shows most affected)
  
Likely Cause:
  - Web scraper parsing error on older pages
  - HTML structure different for historical content
  - Text extraction pulled from wrong element

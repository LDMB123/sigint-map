### Sizing & Scaling

### Icon Sizes
```
Icon Dimensions (all PNG):

Large Format:
  512×512 px    │ High-res displays, app stores
  192×192 px    │ Standard Android, web home screen
  180×180 px    │ Apple iOS touch icon

Scale = Size / 512

Examples:
  512px: scale = 1.0   (base size)
  192px: scale = 0.375 (375% of 512)
  180px: scale = 0.352 (352% of 512)
```

### Component Sizing at 512×512
```
Body Radius:        90 pixels
Head Radius:        70 pixels
Eye Radius:         15 pixels each
Eye Spacing:        28 pixels apart
Horn Height:        100 pixels
Horn Width:         25 pixels
Mane Puff Radius:   22 pixels
Heart Size:         60 pixels
Heart Bumps:        28 pixel radius
Smile Width:        30 pixels

Position Center:
  X-center: 256px
  Y-center: ~230px (slightly higher for proportions)
```

### Scaling Formula
All dimensions scale linearly with size:
```
Actual Dimension = Base Dimension × (Icon Size / 512)

Examples for 192×192:
  scale = 0.375
  Body Radius = 90 × 0.375 = 33.75px
  Head Radius = 70 × 0.375 = 26.25px
  Eye Radius = 15 × 0.375 = 5.625px
```

### Design Principles

### 1. Kawaii Style Elements
- Large eyes with highlights (cute)
- Rounded shapes everywhere (no sharp corners)
- Pastel soft colors (gentle, not aggressive)
- Simplified geometric forms (easy to recognize)

### 2. Age Appropriateness (4-year-old)
- Large touch targets (if interactive)
- Bright but not overwhelming colors
- Simple, clear design (easy to understand)
- Friendly, non-threatening character
- Emphasizes kindness (heart theme)

### 3. Recognition at Small Sizes
- Unicorn silhouette recognizable at 192×192
- Horn distinctive feature
- Heart clearly visible
- Colors distinct at small scales
- No tiny details that disappear

### 4. Child-Friendly Aesthetics
- Smooth curves (no sharp edges)
- Warm color palette
- Symmetrical composition
- Clear focal point (the heart)
- Overall cheerful appearance

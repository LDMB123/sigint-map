## 5-Minute Diagnostic

Run this to assess your PWA:

```javascript
// In browser console
console.log('=== PWA Diagnostic ===');

// 1. Service Workers
navigator.serviceWorker?.getRegistrations().then(regs => {
  console.log(`SWs registered: ${regs.length} (should be 1)`);
  regs.forEach(r => console.log(`  Scope: ${r.scope}`));
});

// 2. Cache Storage
caches.keys().then(names => {
  console.log(`Caches: ${names.length}`);
  names.forEach(n => console.log(`  - ${n}`));
});

// 3. IndexedDB
indexedDB.databases?.().then(dbs => {
  console.log(`IndexedDB: ${dbs.length} databases`);
  dbs.forEach(d => console.log(`  - ${d.name}`));
});

// 4. Manifest
fetch('/manifest.json').then(r => r.json()).then(m => {
  console.log(`Manifest: ${m.name}`);
  console.log(`  Display: ${m.display}`);
  console.log(`  Installable: ${m.icons?.length > 0 ? 'Yes' : 'No'}`);
});

// 5. Offline Status
console.log(`Online: ${navigator.onLine}`);
console.log(`=== End Diagnostic ===`);
```

**If output shows**:
- SWs registered: 1 ✓ | >1 ✗ (use `sw-race-condition-fix`)
- Caches: >0 ✓ | 0 ✗ (use `sw-debugging-checklist`)
- IndexedDB: >0 ✓ (good for offline)
- Manifest: Present ✓ (use `sw-debugging-checklist` to validate)

## Common Copy-Paste Patterns

### Pattern 1: Guard Against Duplicate Registration
```typescript
// From: sw-race-condition-fix.md → Step 3
let initialized = false;

export const pwaStore = {
  async initialize() {
    if (initialized) return;  // Guard
    initialized = true;
    // ... your code
  }
};
```

### Pattern 2: Event Listener with Cleanup
```typescript
// From: sw-memory-leak-detection.md → Step 4
const controller = new AbortController();

$effect(() => {
  navigator.serviceWorker.addEventListener('message', handler, {
    signal: controller.signal  // Cleanup on effect return
  });

  return () => controller.abort();  // Cleanup function
});
```

### Pattern 3: Queue Request for Offline
```typescript
// From: offline-queue-pattern.md → Step 3
const uuid = await queueRequest('/api/data', {
  method: 'POST',
  body: { /* ... */ },
  maxRetries: 3
});
```

### Pattern 4: Proper Port Closure
```typescript
// From: sw-memory-leak-detection.md → Step 6
const port = new MessageChannel();
return new Promise(resolve => {
  const timeout = setTimeout(() => {
    port.port1.close();  // Close on timeout
    resolve(null);
  }, 2000);

  port.port1.onmessage = () => {
    port.port1.close();  // Close on success
    resolve(data);
  };
});
```

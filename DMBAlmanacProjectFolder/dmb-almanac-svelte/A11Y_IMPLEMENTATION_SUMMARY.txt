USAGE GUIDELINES
================================================================================

BEFORE USING COMPONENTS:
1. Read ACCESSIBILITY_GUIDE.md for detailed documentation
2. Review COMPONENT_A11Y_CHECKLIST.md for quick reference
3. Test with keyboard and screen reader (NVDA/VoiceOver)

DROPDOWN USAGE:
- All menu items must have role="menuitem"
- Keyboard support: Arrow keys, Home, End, Escape
- Focus trap within menu when open

TABLE USAGE:
- Always provide caption for context
- Use summary for complex data tables
- Sortable columns properly marked
- Clickable rows announced as buttons

PAGINATION USAGE:
- Current page marked with aria-current="page"
- All buttons have descriptive aria-labels
- Proper navigation semantics

TOOLTIP USAGE:
- Supplementary information only
- Never use for required form instructions
- Works on focus and hover

ERROR BOUNDARY:
- Errors announced immediately
- Clear button for user action
- Proper error context preserved

UPDATE PROMPT:
- Appears as alert dialog
- Focus on primary action
- Keyboard dismissible with Escape

================================================================================
                           TECHNICAL NOTES
================================================================================

WHY THESE OPTIMIZATIONS MATTER ON APPLE SILICON:

M-series Macs (M1/M2/M3/M4) have:
  • Unified Memory Architecture (UMA)
    → GPU and CPU share same memory pool
    → GPU paint operations compete with CPU for bandwidth
  • Metal GPU with CPU-like performance
    → Excels at transform calculations
    → Poor at layout/paint operations
  • ProMotion displays capable of 120Hz
    → 8.33ms per frame budget
    → GPU operations that miss = visible jank

ANIMATION PROPERTIES - GPU STATUS:

  GPU-ACCELERATED (Compositor Thread):
    ✅ transform (translate, scale, rotate, skew)
    ✅ opacity
    ✅ filter (blur, brightness, etc)
    ✅ backdrop-filter

  NOT GPU-ACCELERATED (Main Thread - Slow):
    ❌ background-position (triggers paint)
    ❌ width, height (triggers layout + paint)
    ❌ left, top (triggers layout)
    ❌ margin, padding (triggers layout)
    ❌ font-size, line-height (triggers layout)

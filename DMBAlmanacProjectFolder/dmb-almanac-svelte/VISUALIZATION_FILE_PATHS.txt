DEVELOPMENT WORKFLOW
--------------------
1. Install: npm install (already has d3, d3-sankey, topojson-client)
2. Dev: npm run dev
3. Build: npm run build
4. Visualizations appear at: http://localhost:5173/visualizations

TESTING
-------
All components accept width/height props for testing
Example: <TransitionFlow data={testData} width={500} height={300} />

STYLING
-------
All components use CSS variables defined in root:
--background, --foreground, --color-primary-500, --border-color, etc.

See: src/routes/+layout.svelte for CSS variable definitions

ACCESSIBILITY
--------------
All components include:
- ARIA labels and roles
- Keyboard navigation
- Reduced motion support
- High contrast mode support
- Screen reader support

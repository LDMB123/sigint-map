================================================================================
  TECHNICAL SPECIFICATIONS
================================================================================

Animation Timeline: animation-timeline: view()
- Tied to element visibility in viewport
- No fixed duration (scroll-dependent)
- Automatically progresses as element scrolls into/out of view

Feature Detection: @supports (animation-timeline: view())
- Chrome 115+: Full support, animations active
- Chrome <115: Graceful fallback, static content
- Other browsers: CSS ignored, static content displayed
- No JavaScript errors in any browser

GPU Acceleration: will-change hints
- Properties used: opacity, transform only
- No layout-affecting properties
- Runs on GPU compositor thread
- 60fps on all platforms, 120fps on ProMotion displays

Accessibility: prefers-reduced-motion respected
- All animations disabled when user has motion preference
- Content remains fully visible
- WCAG 2.1 AAA compliant

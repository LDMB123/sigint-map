CACHING STRATEGY SUMMARY
================================================================================

CACHE-FIRST (Use cached, don't check network):
├─ /api/songs              (7 days)
├─ /api/venues             (30 days)
├─ /api/tours              (1 year)
├─ /api/guests             (30 days)
├─ /api/shows/{id}         (1 year)
├─ /api/shows/{id}/setlist (Forever)
├─ Images                  (30 days)
└─ Fonts                   (1 year)

NETWORK-FIRST (Try network, fallback to cache):
├─ /api/shows              (3s timeout, 12hr cache)
├─ /api/liberation         (10s timeout, 12hr cache)
├─ /api/favorites          (5s timeout, 24hr cache)
└─ Navigation              (5s timeout)

NETWORK-ONLY (Never cache):
└─ /api/search             (Always fresh Meilisearch)

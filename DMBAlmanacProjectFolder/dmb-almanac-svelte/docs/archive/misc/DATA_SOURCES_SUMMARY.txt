RISKS & MITIGATION
================================================================================

Risk: HTML structure changes on dmbalmanac.com
  Impact: Scraper failures
  Probability: LOW
  Mitigation: Use robust selectors, implement fallback parsing

Risk: Rate limiting tolerance unknown
  Impact: IP blocking
  Probability: VERY LOW
  Mitigation: Start with 1-2 sec delays, monitor response codes

Risk: Incomplete data on some pages
  Impact: Null values in database
  Probability: MEDIUM
  Mitigation: Handle null gracefully, use Optional fields

Risk: ID extraction difficulties (SID, VID, TID)
  Impact: Can't fetch individual pages
  Probability: MEDIUM
  Mitigation: Parse list pages first to get IDs, validate extraction

HOW IT WORKS
================================================================================

USER JOURNEY:
  1. User visits DMB Almanac home page
     └─ Service Worker already registered
     └─ Manifest already valid
  
  2. Browser checks PWA criteria
     └─ HTTPS/localhost ✓
     └─ Valid manifest ✓
     └─ Service Worker ✓
     └─ Icons present ✓
  
  3. Browser fires beforeinstallprompt event
     └─ InstallBanner component listens
     └─ Captures event in state
     └─ Sets isVisible = true
  
  4. Banner appears on home page
     └─ "Install DMB Almanac" heading
     └─ Description of benefits
     └─ "Install App" button
     └─ "Not now" button
  
  5. User clicks "Install App"
     └─ handleInstallClick() called
     └─ deferredPrompt.prompt() called
     └─ Native install dialog shown by browser
  
  6. User completes installation
     └─ Clicks "Install" in dialog
     └─ Browser installs app
     └─ appinstalled event fires
  
  7. Component detects installation
     └─ Sets isInstalled = true
     └─ Sets isVisible = false
  
  8. Banner automatically hides
     └─ User sees confirmation
     └─ App now on home screen

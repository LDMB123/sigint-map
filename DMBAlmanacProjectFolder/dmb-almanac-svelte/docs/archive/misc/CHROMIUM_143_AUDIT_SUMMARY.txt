   Status: D (Not Implemented)
   Location: /app/my-shows/page.tsx (lines 419-445)

   ✅ Opportunity:
   Process favorites in batches with scheduler.yield()

   Benefit:
   - Main thread remains responsive
   - User can interact while list loads
   - Better perceived performance

   Impact: +35% INP, 0% CLS
   Effort: LOW

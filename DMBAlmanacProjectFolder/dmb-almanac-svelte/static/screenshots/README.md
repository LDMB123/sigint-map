# PWA Screenshots

This directory contains screenshots for the Web App Manifest.

## Required Screenshots

### Desktop (wide)
- `desktop-home.png` - 1920x1080 - Home page showing recent shows and statistics
- `desktop-setlist.png` - 1920x1080 - Show setlist detail view

### Mobile (narrow)
- `mobile-home.png` - 750x1334 - Home page mobile view
- `mobile-songs.png` - 750x1334 - Song database on mobile

## Guidelines

1. Use actual app screenshots with real data
2. Ensure dark theme is used (matches app theme)
3. No personal information should be visible
4. Optimize images for web (PNG, compressed)

## Generation

Screenshots can be captured using:
- Chrome DevTools (F12 > ... > Capture screenshot)
- Playwright for automated capture
- Manual screenshots resized appropriately

================================================================================
KEY CONVERSIONS HIGHLIGHTS
================================================================================

CENTIPEDE NOTATION (Shorthand)
  margin-left: X; margin-right: X;
  → margin-inline: X;

FULL COVERAGE POSITIONING
  top: 0; left: 0; right: 0; bottom: 0;
  → inset: 0;

OPPOSITE SIDES
  left: 1rem; right: 1rem;
  → inset-inline: 1rem;

EXPLICIT BLOCK/INLINE
  margin: 0 0 1rem 0;
  → margin: 0; margin-block-end: 1rem;

INTERNATIONALIZATION BENEFITS
================================================================================

BEFORE (Physical Properties):
  .sidebar { margin-left: 20px; padding-right: 10px; }
  ├─ English (LTR): Works correctly ✓
  └─ Arabic (RTL): Spacing appears on wrong sides ✗

AFTER (Logical Properties):
  .sidebar { margin-inline-start: 20px; padding-inline-end: 10px; }
  ├─ English (LTR): margin-inline-start = left ✓
  ├─ Arabic (RTL): margin-inline-start = right ✓
  └─ Browser handles direction automatically ✓

Result: Single CSS works for all text directions

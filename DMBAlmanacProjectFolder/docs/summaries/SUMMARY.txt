ALL CSS FILES: Media Queries
─────────────────────────────
CURRENT: Verbose min-width/max-width syntax
STATUS: 🟡 Working but outdated syntax
IMPROVEMENT: Use CSS range syntax (Chrome 143+)

Before:  @media (min-width: 640px) and (max-width: 1023px) { }
After:   @media (640px <= width < 1024px) { }

Before:  @media (min-width: 1024px) { }
After:   @media (width >= 1024px) { }

Impact: ⭐⭐ Readability improvement
Time: ~1 hour to update all files
Risk: Zero (purely syntactic change)
